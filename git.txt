一、从github上面拉一个项目下来


	1. 先本地生成 SSH密钥  与github上的密钥对应  

		//本地操作 生成密钥 
		$  ssh-keygen -t rsa -C "youremail@example.com"

			邮箱改成自己的邮箱 然后一直回车

		密钥地址
		c/user/Administrator/.shh/.id_rsa.pub

		.id_rsa  这个是私钥 
		.id_rsa.pub  这个是公钥 github上使用公钥

		再在github上面 New SSH key 

	2. $ git clone 地址  
		
		下载完成就ok




	如果本地已有该项目，以上不走可以不执行，直接

		git pull




二、从本地把项目上传至github
	1. 本地处理  
		1.1 先建立本地仓库

			git init
			注意: 要新建一个 .gitignore  文件  把不需要上传的文件忽略

		1.2 把项目文件放入本地仓库
			git add .  //  . 是把所有的文件添加

		1.3 提交
			git commit -m '备注'


	2. 与远程仓库( github ) 建立连接

		// 建立连接
		git remote add origin git@github.com:ItGethin/BookStores.git

		// 查看连接的状态
		git remote -v


	3. 把项目上传到远成仓库

		git push

		注意:

			★如果是第一次提交 则需要强行提交
		 	git push -u origin master

		 	★如果远程仓库与本地仓库有冲突文件( 远成仓库不是空的) 则需要覆盖
			git push -u origin master -f
